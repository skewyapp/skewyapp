<p>У Skewy две цели:</p> <p>1) При нажатии кнопки Воспроизведение звук Skewy будет маскировать ваш разговор / сделает данные, улавливаемые микрофоном, непригодными для использования (как помеха). В результате у того, кто захочет использовать эти данные, будет практически невозможная задача отфильтровать то, что вы сказали. Вкладка Звук позволяет точно настроить уровень звука и установить дополнительный сигнал, уведомляющий, когда маскировка больше не происходит.</p> <p> 2) Skewy анализирует данные для выявления ультразвуковых (неслышимых) сигналов. Такие сигналы используются рекламными/трекинговыми компаниями, такими как SilverPush, для доступа к вашему телефону незаметно для вас. Skewy может уведомить вас о наличии такого сигнала и по желанию заглушить его, если включен частотный фильтр.</p>
<p>Все данные визуализируются с помощью живых графиков (как по звуку, так и по частоте).</p> <p>Лучше всего он работает с небольшим аппаратным расширением (маленький динамик/наушники), чтобы сфокусировать звук на микрофоне, делая его беззвучным для окружающих. Идею того, как это сделать, можно найти в меню.</p> <p> Никакие реальные звуковые данные не сохраняются.</p>
